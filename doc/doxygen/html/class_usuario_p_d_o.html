Función para validar un usuario

Función para comprobar si existe un usuario dados el código y contraseña; devuelve un objeto Usuario si existe y null si no o en caso de PDOException.

Parameters

Returns
Usuario|null|PDOException Devuelve un objeto usuario si existe. Devuelve null si no existe. Devuelve PDOExcetion en caso de error.
